1.	Название проекта: Волшебный сад

2.	Короткое описание :
Игра представляет собой симулятор садоводства, главным героям является фея, которая помогает своему саду расцветать.

3.	Платформа(ы) и целевые устройства (PC / Android )
4.	Жанр и сеттинг: Cozy Farming Simulation с элементами магии и управления ресурсами.
5.	Игровой цикл (core loop)  - 
События игры разворачиваются в лесу, где находится опушка с грядками, где и будут расти волшебные растения. Главная героиня – лесная фея, которая ухаживает за своим садом, с помощью магии. У феи существуют ее игровые лимиты, а именно – мана. Мана ограничивает возможности феи использовать ее волшебные силы. 

Шаг 1: 
Все грядки пустые и игрок нажав на грядку должен выбрать какое растение посадить хозяйка сада.

Шаг 2:
После выбора растения, фея сажает цветок. Запускается тайминг роста у растения.

Шаг 3:
Игрок может сделать выбор – ждать пока растение вырастит или использовать волшебные силы феи.

Шаг 4: 
Фея применяет свои силы, для ускорения роста растения, что тратит n-количество маны. Мана восстанавливается со временем.

Шаг 5: 
Растение выросло, его уже можно собрать. Игрок направляет фею и та собирает растение, за это пополняется золото.

Шаг 6: 
С помощью золота можно купить новые растения? (дописать реализацию)

6.	Главный персонаж / аватар  - технические требования: кол-во полигонов, LOD, наличие рига и скиннинга
В роли главного персонажа выступает – фея. 
Параметры: Мана
